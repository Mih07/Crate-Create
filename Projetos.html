1- Portfólio

É uma atividade acadêmica da disciplina Aplicação Web e Móveis, onde o objetivo é aplicar os conceitos estudados html, CSS, Javascript(livre), acesso à um provedor para publicar o site. Desenvolver um site, totalmente funcional, com no mínimo quatro links e publicá-lo num servidor de hospedagem gratuito.

O objetivo de desenvolver este site de portfólio é apresentar meus projetos e habilidades de maneira profissional e acessível. Este site serve como uma plataforma centralizada onde potenciais empregadores, clientes e colaboradores podem conhecer meu trabalho e entender minha experiência e competências. Os objetivos principais deste site são:

Demonstração de Habilidades Técnicas

* Desenvolvimento Web: Mostrar minha capacidade de criar sites responsivos e modernos utilizando HTML5, CSS3 e JavaScript.
* Design e UX: Apresentar habilidades de design e experiência do usuário, criando uma interface intuitiva e visualmente atraente.

Centralização de Projetos

* Portfólio de Projetos: Reunir todos os meus projetos em um único local, facilitando o acesso e a visualização de meu trabalho.
* Detalhes de Projetos: Fornecer descrições detalhadas de cada projeto, incluindo objetivos, tecnologias utilizadas e desafios superados.

Facilitar a Comunicação

* Contato Profissional: Disponibilizar formas de contato direto para empregadores e clientes potenciais, como e-mail e redes sociais profissionais.
* Blog e Atualizações: Manter uma seção de blog ou atualizações para compartilhar conhecimentos, novidades e experiências na área de desenvolvimento.

Estabelecimento de Presença Online

* Marca Pessoal: Construir uma marca pessoal forte e consistente, apresentando-me de maneira profissional e autêntica.
* SEO e Visibilidade: Implementar práticas de SEO para aumentar a visibilidade do site nos motores de busca, atraindo mais visitantes e oportunidades.

Desenvolvimento Contínuo

* Evolução do Portfólio: Adicionar novos projetos e atualizações conforme minha carreira progride, mantendo o portfólio sempre atualizado.
* Aprendizado Contínuo: Utilizar o site como um espaço para experimentar novas tecnologias e tendências no desenvolvimento web.

2- Tela de Login


O objetivo de desenvolver esta tela de login é demonstrar minhas habilidades em design de interfaces de usuário e desenvolvimento front-end, criando uma interface intuitiva e responsiva para autenticação de usuários. Este projeto mostra minha capacidade de trabalhar com HTML5, CSS3 e JavaScript para construir componentes de interface essenciais. Os principais objetivos deste projeto são:


Design de Interface de Usuário (UI)

* Interface Amigável e Intuitiva: Criar uma tela de login que seja fácil de usar e visualmente agradável, proporcionando uma boa experiência para o usuário.
* Design Responsivo: Garantir que a tela de login funcione bem em diferentes dispositivos e tamanhos de tela, utilizando técnicas de design responsivo.

Interatividade e UX

* Validação de Formulários: Implementar validações no lado do cliente para garantir que os usuários forneçam informações corretas antes de submeterem o formulário.
* Feedback ao Usuário: Fornecer feedback visual e mensagens claras para guiar o usuário durante o processo de login, incluindo mensagens de erro e sucesso.

Melhores Práticas de Desenvolvimento

* Código Limpo e Organizado: Escrever código HTML, CSS e JavaScript de maneira clara e organizada, seguindo as melhores práticas de desenvolvimento web.
* Acessibilidade: Assegurar que a tela de login seja acessível a todos os usuários, incluindo aqueles com deficiências, utilizando boas práticas de acessibilidade web (ARIA).

Estilização e Animação

* CSS Moderno: Utilizar CSS3 para estilizar a tela de login, incluindo uso de flexbox/grid, animações e transições suaves para melhorar a experiência do usuário.
* Consistência Visual: Manter uma consistência visual com o restante do site ou aplicação, utilizando uma paleta de cores e tipografia adequada.

Preparação para Integração com Black-End

* Estruturação para Futura Integração: Preparar a tela de login de maneira que possa ser facilmente integrada com uma lógica de back-end no futuro, utilizando boas práticas de estruturação de código.

Tecnologias Utilizadas

* HTML5 para a estruturação do conteúdo.
* CSS3 para estilização e design responsivo.
* JavaScript para validação e interatividade.

3- Jogo da Velha


Funcionalidades

Criei um jogo da velha interativo como parte do meu portfólio de desenvolvimento web. Este projeto demonstra minhas habilidades em HTML, CSS e JavaScript, além de conceitos de lógica de jogo e interatividade com o usuário.


* Interface Responsiva: Utilizei HTML5 e CSS3 para criar uma interface amigável e responsiva, que se adapta a diferentes dispositivos e tamanhos de tela.
* Lógica de Jogo: Desenvolvi a lógica de jogo em JavaScript, garantindo que o jogo detecte automaticamente as vitórias, empates e movimentos inválidos.
* Interatividade com o Usuário: Implementação de eventos de clique para permitir que os jogadores façam suas jogadas interagindo diretamente com o tabuleiro.
Reinicialização do Jogo: Incluí uma funcionalidade para reiniciar o jogo a qualquer momento, permitindo partidas consecutivas sem recarregar a página.
Indicador de Jogador: Um indicador visual para mostrar de quem é a vez de jogar.

Desafios e Soluções

* Lógica de Vitória: Um dos desafios foi implementar a verificação de vitória de maneira eficiente. Resolvi isso utilizando arrays para representar o tabuleiro e checando todas as possíveis combinações de vitória após cada movimento.
* Experiência do Usuário: Trabalhei na melhoria da UX, garantindo que o jogo fosse intuitivo e visualmente agradável. Realizei testes com usuários para refinar a interface.

Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

Aprendizados e Melhorias Futuras

* Este projeto me proporcionou uma compreensão mais profunda de manipulação do DOM e lógica de jogo. Pretendo expandi-lo no futuro, implementando funcionalidades como um modo de jogo contra a IA, usando o algoritmo Minimax para tornar o oponente mais desafiador.
